Steps on how to install Visual Basic 6.0

1. Use the link provided in the group or use Download Visual Basic 6.0 (it's the same as the one in the group).

2. No matter what you see e.g. 'Couldn't preview file...' just click on download.

3. Next you may see "Google can't scan this file for viruses...." just click download anyway.

4. If you used a phone to download the file, transfer it to your computer via a usb cable or other methods.

5. Go to the folder where you transferred it to (or downloaded it) on your laptop and right click on it, select 'extract all' in the dropdown menu.

6. Open the folder you just extracted.

7. Open the subfolder inside the extracted folder.

8. Open the sub folder again to see the ISO folder and open the ISO folder

9. Now double-click on the VBP600ENU1 file (this mounts the disk image file on your system)

10. You should see something like this in your file explorer, this shows that the disk image has been mounted. Open the new drive that represents the mounted disk image.

11. Select all the files and folders and copy them.

12. Navigate to the 'Desktop' folder, create a new folder (I named mine 'VB6_Setup' you can name yours anything you want as long as it's a valid folder name), and paste the previously copied files and folders here.

13. You should have something like this now.

14. Select setup.exe and right-click on it (or just setup like as in here). Make sure the setup you're clicking on has the VB 6.0 icon (the one my cursor is highlighting), as there 2 setup files; one is an application file while the other is a configuration settings file. After right-clicking select 'Run as Administrator' in the dropdown menu.

15. Select Yes on the UAC window.

16. On the Program Compatibility Assistant window, select 'Run the program without getting help' (Windows 10), or select 'Cancel' (Windows 11).

17. Click on 'Next'.

18. Click on 'I accept the agreement' and click on 'Next'.

19. Enter what I entered for the product ID number (three 1s and seven 1s). For your name and company name enter whatever you want. The name field may be filled automatically you can decide to change it or leave it as it is. Click on 'Next' when you're done.

20. Select Visual Basic 6.0 and click 'Next'.

21. You can choose a custom installation path but it is recommended to use the default installation path.

22. You'll see a window telling you to wait; wait for it, after that you'll see the window below, click on 'Continue'.

23. You can copy this if you want but I don't think it's necessary. Click on 'Ok'.

24. Wait for some time.

25. Click on 'Custom', never use 'Typical' as the installation will hang and not complete if you do.

26. Click on 'Data Access' to uncheck it.

27. Click 'OK' on the warning prompt.

28. Click on 'Change Options'.

29. Click on 'ADO, RDS, and OLE DB Providers' to uncheck them as they will hang your installation and cause it to fail.

30. Click on 'OK' in the warning prompt.

31. Click on 'OK' in the main dialog window.

32. Click on 'Continue' to proceed with the installation.

33. Wait for the installation to complete.

34. Select 'Restart windows' (make sure you don't have any pending work you haven't saved on your computer as they'll likely be lost).

35. Click 'Exit'

36. Click 'Yes'.

37. If you see this window, just click 'OK' and close all the Program Compatibility windows.
